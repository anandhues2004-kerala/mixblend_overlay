# 🌐 Tailwind CSS Hero Section Project

## 📌 Overview

This project is a simple and modern **full-screen hero section** built using **HTML** and **Tailwind CSS**. It features a background image with a color overlay and centered text content.

The design focuses on clean layout, readability, and responsiveness.

---

## 🚀 Features

* ✅ Full-screen background image
* ✅ Smooth color overlay for better text visibility
* ✅ Centered content using Flexbox
* ✅ Responsive typography
* ✅ Clean and minimal UI design

---

## 🛠️ Technologies Used

* **HTML5**
* **Tailwind CSS (CDN version)**

---

## 📂 Project Structure

```
project-folder/
│── index.html
│── README.md
```

---

## 💻 How to Use

1. Copy the HTML code into a file named `index.html`
2. Open the file in your browser
3. Make sure you have an internet connection (Tailwind CDN is used)

---

## 🎨 Customization

You can easily modify:

* **Background Image** → Replace the image URL
* **Overlay Color** → Change `bg-blue-500/60` to any Tailwind color
* **Text Content** → Edit heading text
* **Font Size** → Adjust `text-3xl`, `md:text-5xl`
